MC ToolBox

*Description:

  • Split: Split the file into smaller files regarding the number of rows.
  • Extract: Extract domain from Website. Company and Website columns' names are compulsory in the input file.
  • Match: Match company emails with the equivalent websites. Company, Website and Email columns' names are compulsory in the input file.
  • End to End: The tool will automatically extract domains and find bulk emails through Snovio.io API. Company and Website columns' names are compulsory in the input file.

    • *Maybe you dont know this:

  • Lowercase or uppercase columns' names do not matter: Company, COMPANY, company, CoMpAnY,... are all accepted and recognizable.
  • You don't have to add all 3 columns' names: Company, Website and Email to use End2End. Only Company and Website are required.

    • *Notes:

  • The Unsuccess file contains data that is failed to fetch emails from Snovio due to server error. If it exists in the output zip, you can put it back to E2E to get the remaining emails. This new feature helps save the account credits when processing large data.
  • With the columns that contain date-time information, the format may be affected.
  • The system will automatically filter out the line which has no information (no website or no email) or is duplicated. Therefore, the number of output data may not be the same as the input file.

    • CSV(.csv) or Excel (.xlsx) only
